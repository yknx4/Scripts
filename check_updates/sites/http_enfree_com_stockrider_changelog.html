PRIME Kernel

데이터 절약 모드 사용 시 주의사항
크롬 또는 순정(or 오페라) 브라우저의 데이터 절약 모드를 사용하면 파일이 손상되고 TWRP에서 플래싱 되지 않습니다.
반드시 해제 후 다운로드 하세요.

PRIME 커널은 더 나은 커널 개발을 위해 기부를 받고 있습니다.
기부는 간접적인 개발 참여 방법입니다.

     기부로 개발에 참여하기
